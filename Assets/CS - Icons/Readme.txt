--------------------------
       How to use
--------------------------

Create something you want to use an icon on (such as 2D Object > Sprite or UI > Button) and then then set one of the icons as the sprite. 

--------------------------
Icons, sizes, and variants
--------------------------

    The following icons are present:
     • Audio: Low, High, Muted, Muted 2
     • Arrows: Left, Right, Up, Down, Top Left, Top Right, Bottom Left, Bottom Right
     • Video: Pause, Rewind (or Restart), Forward (or Restart 2)
     • Messages: Question (?), Warning (!), Checkmark, Close/Quit
     • Misc: About, Camera, Diamond (filled), Diamond (outline), Home, Minus, More, More 2, Plus, Settings, Shop (or Money)
	   In total: 30 icons
	
    The icons are available in the following variants:
     • Normal (bright)
     • Hover (slightly darker)
     • Pressed (dark)
     • Unavailable/Active (slightly brighter)

    The icons are in the following colors:
     • Blue
     • Green
     • Grey
     • Red
     • Teal
     • Yellow
	 
    The icons are available in 512px (diffrrent resolutions available by exporting the SVG file in Inkscape), and have antialising.
